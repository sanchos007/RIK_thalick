Щоденник настрою 
Мобільний додаток на React Native (Expo) для трекінгу настрою та перегляду статистики. Розроблено в рамках залікової роботи.

🚀 Основні функції
Додавання записів настрою (Емодзі + Текстова нотатка).

Історія записів із можливістю фільтрації (Тиждень, Місяць, Всі) та видалення.

Статистика у вигляді кругової діаграми.

Налаштування з можливістю очищення даних.

Локальне збереження через AsyncStorage.

🛠 Технології та Бібліотеки
React Native & Expo

TypeScript

React Navigation (Bottom Tabs)

@react-native-async-storage/async-storage - для локального збереження

react-native-chart-kit - для графіків

expo-linear-gradient - для сучасного UI

@expo/vector-icons - для іконок

uuid - для генерації унікальних ID записів

📦 Інструкція із запуску
Клонуйте репозиторій:

Bash
git clone https://github.com/sanchos007/RIK_thalick.пше

Встановіть залежності:

Bash
npm install
Запустіть проєкт через Expo:

Bash
npx expo start
Відскануйте QR-код за допомогою додатку Expo Go на вашому iOS або Android пристрої.
